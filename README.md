Présentation
Ce projet consiste à développer une API REST complète
Fonctionnalités principales :
* Authentification sécurisée (JWT + bcrypt)
* Gestion des utilisateurs avec rôles
* CRUD des ouvrages et catégories
* Gestion du panier et des commandes
* Système d'avis et commentaires validés
* Listes de cadeaux partageables

Technologies utilisées
Node.js + Express
MySQL

bcrypt (sécurité mots de passe)
